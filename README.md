# safinalo-project
ეს არის საიტი საკვების დელივერზე, რომელიც მაქსიმალურად მორგებულია მომხმარებლზე
ფუნქციონალი რომელიც დავამატე:
Sticky Navigation: მენიუ, რომელიც მოყვება სქროლს და იცვლის ვიზუალს უკეთესი ნავიგაციისთვის
Smooth Scrolling: რბილი გადასვლები საიტის სხვადასხვა სექციებს შორის (Home, About, Menu, Contact)
Booking System Alert: ინტერაქტიული შეტყობინება "Book A Table" ღილაკზე დაჭერისას.
Responsive Design: საიტი სრულად მორგებულია პლანშეტებსა და მობილურ მოწყობილობებზე.
Scroll Animations: ელემენტების ლამაზი გამოჩენის ეფექტები (Fade-in) გვერდის ჩამოწევისას
CSS Reset: გამოყენებულია reset.css ბრაუზერებს შორის სტილების უნიფიკაციისთვის.
